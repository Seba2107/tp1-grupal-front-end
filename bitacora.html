// proceso

BITÁCORA

Semana 1
Inicio del proyecto

Definimos el nombre del equipo, la estructura de archivos y el enfoque visual. Cada integrante aporta su estilo propio, luego unificamos los estilos y desarrollamos la portada común.

Decisiones de diseño
Diseño unificado con personalidad

Elegimos una estética oscura, tipografías modernas y tarjetas con efecto glowy para mantener la coherencia entre la página principal y las tarjetas individuales. Cada integrante conserva su propia paleta dentro del mismo universo visual.

Dificultades encontradas
Responsive y consistencia

El mayor desafío fue adaptar la estructura del sitio a varios tamaños de pantalla y mantener las tarjetas individuales legibles. Resolvímos esto con breakpoints claros y ajustes en el tamaño de fuentes, tarjetas y el comportamiento de hover.

Cambios importantes
Separación de la bitácora

Inicialmente la bitácora estaba sólo en la sección del index, pero se decidió crear un archivo independiente para cumplir con el requerimiento y mejorar la navegación. También desplazamos estilos específicos a CSS para limpiar el HTML.

Implementación técnica
Contenido obligatorio y mejoras

Agregamos los campos requeridos en las tarjetas personales: foto, nombre, ubicación, edad, habilidades, películas y discos. Además, mejoramos la presentación con iconos y un hover visual para las películas.

Reflexión
Plan para TP2

Esta bitácora se ampliará en el próximo Trabajo Práctico con reflexiones más profundas sobre el proceso, las decisiones técnicas y la evolución del proyecto.
